# Calculate APY

Update the below values in the code before running the script.
const SDAO_USD = 1.9284; // USD value of the SDAO token
const SDAO_STAKE_YEARLY_REWARDS = '1214720'; // Yearly rewards

## Install the dependencies

```
yarn
```

## Compile code

```
yarn build
```

### Run the script

```
node dist/
```
